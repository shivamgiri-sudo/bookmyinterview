# Developer Sprint Plan

## Sprint 1: Foundation
Frontend shell, backend FastAPI, routing, design tokens, API contract.

## Sprint 2: Client Intake + JD
Smart intake, gap detection, JD generation, assessment path preview.

## Sprint 3: Assessment Engine
Rules matrix, dependency scores, client override, role-wise assessment paths.

## Sprint 4: Trait Engine
Trait matrix, question generation, validation, approval lifecycle, scoring rubric.

## Sprint 5: Candidate Portal
Resume upload, parsed profile, assessment journey, scorecard, slot booking.

## Sprint 6: Audio/Video
Recording UI, storage adapter, STT adapter, transcript evaluation, video evaluation.

## Sprint 7: Admin Console
Role taxonomy, question bank, rules, audit logs, compliance settings.

## Sprint 8: Integrations
Calendar, email, WhatsApp, ATS, payment, MCP servers.
