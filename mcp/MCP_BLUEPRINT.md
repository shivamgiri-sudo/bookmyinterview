# MCP Blueprint for BOOK MY INTERVIEW

## Design MCP
- Figma MCP: read components, variables, frames; sync design tokens.
- Asset MCP: brand icons, illustrations, motion specs.

## Developer MCP
- Cursor MCP: expose product docs, acceptance checklist, design system, API contract.
- GitHub MCP: issues, PRs, CI, code review checklist.

## Product MCP Servers
- Client CRM MCP: company, contracts, hiring history.
- Candidate DB MCP: candidate profile, assessment history, consent.
- JD Intelligence MCP: templates, gap rules, skill/trait matrix.
- Assessment MCP: path builder, questions, audio/video, scorecards.
- Calendar MCP: slots, bookings, reschedule.
- Communication MCP: email, WhatsApp, SMS, reminders.
- Compliance MCP: audit logs, consent, protected attribute safeguards.
